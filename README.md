# SnakeInConsole
A simple console-based Snake game where you guide a growing snake to eat food, avoid walls, and fill the entire field to win.

# Консольная игра «Змейка» на Python  

## Описание проекта  
Данная игра создана как учебный проект для закрепления базовых навыков программирования на **Python**.  
Основная цель разработки – показать, как можно реализовать простую игру без использования графических библиотек, работая только с консолью.  

Проект подойдёт для начинающих программистов, которые хотят понять:  
- как устроен игровой цикл;  
- как обрабатывается ввод с клавиатуры;  
- как реализовать движение, столкновения и подсчёт очков.  

Игра написана полностью на **чистом Python**, без внешних зависимостей, и запускается в стандартной консоли любой операционной системы.
